# QR_generator
A simple Python utility that generates QR code images from user-provided text or URLs.

## 기능

- 텍스트나 URL을 QR 코드로 바꿔줘요!

---

## 터미널에서는 이런 형태로 나타나요 (예시)

QR코드로 만들 텍스트나 URL을 입력하세요 : 

 QR 코드가 성공적으로 생성되었습니다.
파일 이름 : my_qrcode.png

---

## 사용법

1. 이 레포지토리를 클론하거나 다운로드
2. Python 설치
3. 터미널에  python qr_generator.py를 입력 후 게임 진행

```bash
 python qr_generator.py
```
